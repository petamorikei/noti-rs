## Project Overview

## Build Commands

```bash
# Build the project
cargo build

# Build release version
cargo build --release

# Run the project
cargo run

# Run tests
cargo test

# Run a specific test
cargo test test_name

# Check code without building
cargo check

# Format code
cargo fmt

# Run linter
cargo clippy
```
